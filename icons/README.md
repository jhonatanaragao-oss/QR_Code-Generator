# Ícones das ferramentas

Coloque os arquivos `.ico` desta pasta e linke cada ícone no cadastro da ferramenta em `script.js`.

Exemplo:

```js
{
  name: "QR Code avançado",
  category: "Gerar",
  description: "Link, WhatsApp, e-mail, localização, Wi-Fi, Pix, vCard e texto livre.",
  initials: "QR",
  icon: "icons/qr-code.ico"
}
```

Se `icon` não existir, o card continua mostrando as iniciais.

Tamanho exibido no card: `24x24` dentro do quadrado atual de `42x42`.
